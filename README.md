# Mobile-Sales-Dashboard
This repository contains interactive data analysis and visualization projects created using Power BI and R Programming.
# 📊 Power BI Project

An interactive Power BI dashboard designed to analyze mobile sales performance, customer ratings, revenue trends, and brand-wise insights.

## 🚀 Features
- Brand-wise Sales Analysis
- Revenue & Profit Insights
- Customer Ratings Analysis
- Interactive Filters & Slicers
- Data Cleaning using Power Query
- R Visual Integration in Power BI

## 🛠 Technologies Used
- Power BI
- DAX
- Power Query
- R Programming
